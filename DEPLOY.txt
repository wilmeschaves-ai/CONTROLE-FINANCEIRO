# ⚠️ PROBLEMAS COMUNS

→ Nenhuma alteração foi feita

## ❌ "not a git repository"

→ Rode:

## ❌ erro banco.py

→ Verificar se arquivo está no GitHub

## ❌ dados sumindo

→ SQLite não persiste online (normal)
